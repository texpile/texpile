Most of what's in this folder is heavily inspired by the [LaTeX Workshop](https://github.com/James-Yu/LaTeX-Workshop) VS Code extension's IntelliSense (completion, hover, folding, go-to-definition), reimplemented for CodeMirror. Data comes from unified-latex's curated CTAN signature DB plus LaTeX Workshop's vendored data: its default macro set (`data/lwMacros.ts`), the tex.json primitives/math set (`data/texMacros.ts`), the per-package keyval files (`data/packages/`, originally from TeXStudio's cwl completion files), the package/class name lists, at-suggestions, and bib entry types. Licenses and provenance: `data/VENDORED-LICENSES.md`; credit also lives in the root README's Acknowledgements.
